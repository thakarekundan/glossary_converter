Glossary creation using NLP

“Glossary is an important part of any book; it makes a book complete.”

A glossary, also known as vocabulary, is a list of alphabetically well-arranged words. A glossary is also known as a lexicon, wordbook. Glossary plays an important role as a tool to learn new words. This project discusses the result of the work done in development of GLOSSARY CREATION. In this project we demonstrate a prototype solution in which we built a dedicated platform for students, or high-tech readers. The aim is to enable them to reach their requirements using a convert feature.
